Комментарии к странице (по работе таблицы)

  • Главная

Для реализации иерархической структуры таблицы используются 2 скрипта:

  • jquery-treetable - Иерархия таблицы
  • gdx-table - сворачивание ячеек таблицы, которые не помещаются на экране на малых разрешениях

Сначала инициализируется *jquery-treetable*, затем *gdx-table*

На больших разрешениях, когда все ячейки помещаются, отображается только иерархическая структура. Если таблица не помещается в размеры экрана, то часть ячеек скрывается. Иконки "стрелки" отображаются, если есть дочерние элементы. На малых разрешениях, при клике на стрелку открываются скрытые ячейки и дочение элементы. Если дочерних элементов у узла нет, то отображается иконка "+" и открываются скрытые ячейки.

Для реализации иерархической структуры таблицы, необходимо установить для tr data-атрибуты см. документацию к плагину.
